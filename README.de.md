## Was ist eine ITIN?

Eine **ITIN (Individual Taxpayer Identification Number)** ist eine US-Steuernummer für Personen, die in den USA steuerpflichtig sind, aber keine Sozialversicherungsnummer (SSN) erhalten können. Sie wird benötigt für:

- **US-Einkommensteuererklärung** (Formular 1040-NR) für Dividenden, Honorare oder Mieteinnahmen aus den USA
- **US-Immobilieninvestitionen** — FIRPTA-Compliance und Erstattung der Quellensteuer
- **US-LLC oder Kapitalgesellschaft** — als ausländischer Eigentümer oder Gesellschafter
- **YouTube / Google AdSense** — Reduzierung der US-Quellensteuer von 30 % auf den Abkommenssatz (oft 0–15 % nach DBA Deutschland–USA)
- **US-Bankkonten oder Investmentkonten** als Nicht-Resident

## Häufige Anwendungsfälle für Deutsche

### Doppelbesteuerungsabkommen (DBA) USA–Deutschland
Das DBA zwischen den USA und Deutschland sieht reduzierte Quellensteuersätze vor. Mit einer ITIN können Sie:
- Dividenden aus US-Aktien mit 15 % statt 30 % versteuern
- Lizenzgebühren und Honorare optimiert abrechnen
- US-Erstattungen effizient beantragen

### US-Immobilien (FIRPTA)
Beim Kauf oder Verkauf von US-Immobilien ist eine ITIN für die FIRPTA-Erklärung erforderlich. Ohne ITIN behält der Käufer 15 % des Kaufpreises ein.

### US-LLC oder GmbH-Äquivalent
Viele deutsche Unternehmer gründen eine US-LLC für den amerikanischen Markt. Eine ITIN ermöglicht die korrekte Steuererklärung und Kontoeröffnung.
