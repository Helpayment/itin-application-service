## ITIN คืออะไร?

**ITIN (Individual Taxpayer Identification Number)** คือหมายเลขประจำตัวผู้เสียภาษีที่ออกโดย IRS สำหรับบุคคลที่มีภาระภาษีในสหรัฐฯ แต่ไม่มีสิทธิ์ขอ Social Security Number (SSN) จำเป็นสำหรับ:

- **การยื่นภาษีของสหรัฐฯ** (แบบฟอร์ม 1040-NR) — เงินปันผล ค่าลิขสิทธิ์ หรือรายได้ค่าเช่าจากแหล่งในสหรัฐฯ
- **การลงทุนในอสังหาริมทรัพย์สหรัฐฯ** — การปฏิบัติตาม FIRPTA และการขอคืนภาษีหัก ณ ที่จ่าย
- **การจดทะเบียน LLC หรือบริษัทในสหรัฐฯ** ในฐานะเจ้าของชาวต่างชาติ
- **YouTube / Google AdSense** — ลดภาษีหัก ณ ที่จ่ายของสหรัฐฯ จาก 30% เป็นอัตราตามอนุสัญญาภาษี
- **เงินปันผลจากหุ้นสหรัฐฯ** และรายได้ passive อื่นๆ

## ทำไมต้องใช้ตัวแทนรับรองจาก IRS (CAA)?

ในฐานะ **ตัวแทนรับรองจาก IRS (CAA)** เราสามารถยืนยันตัวตนของคุณได้โดยตรงโดยไม่ต้องส่งพาสปอร์ตต้นฉบับไปยัง IRS

✅ ไม่ต้องส่งพาสปอร์ตไปสหรัฐฯ  
✅ จัดเตรียมและตรวจสอบแบบฟอร์ม W-7  
✅ ยื่นตรงต่อ IRS  
✅ ยื่นซ้ำฟรีหากถูกปฏิเสธ  
✅ 100% ออนไลน์ — ไม่ต้องไปสถานทูตสหรัฐฯ

## ขั้นตอนการสมัคร

1. **ส่งเอกสารออนไลน์** — อัปโหลดพาสปอร์ตและเอกสารภาษี
2. **ตัวแทน CAA ตรวจสอบ** — เราจัดเตรียมแบบฟอร์ม W-7
3. **ยื่นต่อ IRS** — ตัวแทนยื่นโดยตรงแทนคุณ
4. **รับ ITIN** — IRS ส่ง ITIN ทางไปรษณีย์ (ประมาณ 6–8 สัปดาห์)
